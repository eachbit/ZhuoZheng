# Chapter01 UI Fonts

Use these fonts for Chapter01 UI only.

## Chosen fonts

- Title / panel title: `SourceHanSerifSC-SemiBold.otf`
- Body / button / prompt / option: `SourceHanSansSC-Medium.otf`
- Number / highlight / counter: `SourceHanSansSC-Bold.otf`

## Unity usage

1. Import these `.otf` files into Unity.
2. Create TMP Font Asset files from the same source files.
3. Reuse the same TMP assets for all Chapter01 UI.
4. Do not replace them with system fonts.

## Suggested mapping

- Objective bar: Sans Medium
- Interaction prompt: Sans Medium
- Dialogue title: Serif SemiBold
- Dialogue body: Sans Medium
- Water choice title: Serif SemiBold
- Water choice content: Sans Medium
- Result banner: Sans Bold
- Puzzle panel title: Serif SemiBold
- Counter / page number: Sans Bold

## Source

- Source Han Sans SC `2.005R`
- Source Han Serif SC `2.003R`
