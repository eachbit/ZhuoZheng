Chapter 01 UI / 园林正式版

第一章整套 UI 方向草图

这次我把草图画得更明显了，不再是暗色占位块。下面每张卡片里都能直接看到：残页计数、目标栏、结果提示、对话框、水流选择框、暗闸校准框这些模块的位置和气质。

你现在不用纠结技术，只要先看“哪一套最像你心里这款游戏的正式 UI”。选定气质后，我再继续拆成具体实现方案。

我的推荐

  • 优先推荐 方案 B：园林匾额 + 墨底金边
  • 它最像正式游戏 UI，不像临时样式
  • 白天、夜景、下雨这三种场景里都更稳
  • 后面你继续加动画、图标、纹样，也不会推翻重做
方案 A

宣纸卷轴感

偏米白宣纸、木框、题签感，像在园林志册上翻章节。文气最强，最传统。

优点：最国风、最书卷。
代价：亮背景下需要额外压暗或描边，才能保证清晰度。
方案 B / 推荐

园林匾额 + 墨底金边

深墨底、暖金线、青绿点缀，像被灯影照亮的匾额与窗棂。正式感最强。

优点：可读性最好，最适合做成完整游戏 UI。
代价：少一点纸面书卷感，但更稳、更耐看。
方案 C

戏台暖木感

暖木、金箔、灯影感更强，像园中戏台与廊下陈设，章节戏剧性最足。

优点：辨识度强，记忆点很高。
代价：如果全程都用，会比另外两套更“热闹”。
你现在可以直接选：A / B / C。
如果你想混搭，也可以直接说：以 B 为主，借一点 A 的宣纸感。
