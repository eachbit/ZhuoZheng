# ZhuoZhengYuan

Unity project for the Zhuozheng Garden gameplay prototype.

## Collaboration

- Use branch-based development. Do not develop directly on `main`.
- Avoid editing the same `.unity` scene file in parallel whenever possible.
- Keep large environment or UI changes in separate branches and merge them back after testing.
- Runtime save data is local only and is ignored by Git.

## Unity Project Settings

- Version Control: `Visible Meta Files`
- Asset Serialization: `Force Text`
